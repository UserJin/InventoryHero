## 고찰
- 일단 필요한 기능을 적절한 스크립트에 구현한다는 생각으로 구현했는데, 그러다보니 GameManager에는 아무런 기능이 없음
- 실제 인벤토리와 인벤토리 UI를 나누어 구성하였는데, 이로 인해 서로 간의 엮인 부분이 많아서 의존성을 최대한 줄이면서 구현하는게 힘들었다. 아직도 불필요하게 의존하는 부분이 존재한다고 생각해서 한 번 다시 구조를 생각해봐야 할것 같다.
